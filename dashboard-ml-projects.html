Proyectos de Machine Learning: Predicción de Retención de Clientes

Resumen del Proyecto

Este proyecto implementa un modelo de Machine Learning para predecir la probabilidad de que un cliente abandone un servicio (churn). Utilizando datos demográficos, de uso del servicio e historial de pagos, el modelo ayuda a identificar clientes en riesgo para que las empresas puedan intervenir proactivamente y mejorar las estrategias de retención. La predicción temprana es crucial para minimizar la pérdida de ingresos.

El objetivo principal es construir un modelo robusto y explicable que no solo prediga, sino que también ofrezca insights sobre los factores que más influyen en la decisión de un cliente de irse.

Metodología y Herramientas

Para este proyecto, se aplicó un enfoque integral de Machine Learning, incluyendo:

  • **Preprocesamiento de Datos:** Manejo de valores faltantes, codificación de variables categóricas, escalado de características.
  • **Selección de Características:** Uso de técnicas como RFE (Recursive Feature Elimination) y análisis de importancia de características.
  • **Modelado:** Se exploraron varios algoritmos, incluyendo Logistic Regression, Random Forest y Gradient Boosting Machines (XGBoost), siendo este último el de mejor rendimiento.
  • **Evaluación:** Métricas como Accuracy, Precision, Recall, F1-Score y ROC-AUC fueron utilizadas para validar el modelo.

Las herramientas principales utilizadas fueron Python, Scikit-learn, Pandas, NumPy y XGBoost.
