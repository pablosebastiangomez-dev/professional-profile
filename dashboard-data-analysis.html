Análisis Exploratorio de Datos: Ventas de Videojuegos Globales

Resumen del Proyecto

Este proyecto se enfoca en el análisis de un conjunto de datos global de ventas de videojuegos, explorando tendencias de la industria, rendimiento por plataforma y editor, y el impacto de géneros específicos a lo largo del tiempo. El objetivo es descubrir patrones ocultos y proporcionar información valiosa para la toma de decisiones estratégicas en la industria del gaming.

Se utilizaron técnicas de limpieza de datos, análisis estadístico y visualización para extraer conclusiones significativas.

Visualización Clave

A continuación, se presenta una visualización interactiva que muestra las ventas globales por género a lo largo de los años. Puedes pasar el ratón sobre los elementos para ver detalles específicos.

Metodología y Herramientas

El análisis se realizó utilizando Python con librerías como Pandas para manipulación de datos, Matplotlib y Seaborn para visualización estática, y Plotly para gráficos interactivos. El proceso incluyó:

  • Limpieza y preprocesamiento de datos (manejo de valores nulos, estandarización).
  • Análisis de tendencias temporales y geográficas.
  • Segmentación de datos por plataforma, editor y género.
  • Identificación de los juegos más vendidos y sus características.
