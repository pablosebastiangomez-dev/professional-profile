# Portfolio Profesional - CTO & Principal Web3 Architect

Este repositorio contiene el código fuente de mi portfolio profesional. El sitio web está diseñado para reflejar mi perfil como **CTO & Principal Web3 Architect** con enfoque en productos de Inteligencia Artificial e Infraestructura de Datos de alto rendimiento.

## 🚀 Arquitectura y Proyectos Destacados

*   **E4C (Education For Culture)**: Plataforma de incentivos EdTech gamificada construida sobre la red **Stellar**. Diseñé e implementé el motor de liquidación automatizada B2B usando **Stellar Disbursement Platform (SDP)**, procesando pagos en stablecoins (USDC), y utilicé **Pollar SDK** para abstracción de cuentas (onboarding de menores de edad con su Gmail escolar).
*   **Asistente de IA en el Aula**: Asistente privacy-first para la automatización pedagógica, el cual procesa el dictado oral de docentes en lenguaje natural y genera pipelines de datos inmutables de presentismo en la blockchain.
*   **NewsChain (Truth Tech Initiative)**: Arquitectura descentralizada para la trazabilidad y prueba de autenticidad en el periodismo digital, implementando capas de inmutabilidad para el registro de procedencia de la información.

## 🛠️ Matriz de Habilidades Técnicas

## 📊 Demos y Dashboards Interactivos

El sitio web integra dashboards interactivos y herramientas financieras en tiempo real:
1.  **Dashboard de Criptomonedas**: Seguimiento en tiempo real con datos de la API de CoinGecko.

## 💻 Configuración Local

### Requisitos Previos

Asegúrate de tener instalado [Node.js](https://nodejs.org/).

### Pasos para Ejecutar

1. Instalar las dependencias del proyecto:
   ```bash
   npm install
   ```

2. Iniciar el servidor local de desarrollo:
   ```bash
   npm start
   ```

3. Abrir el navegador en la dirección local indicada por `http-server` (usualmente `http://localhost:8080`).

## 🌐 Despliegue

El proyecto está optimizado y listo para ser desplegado de forma estática en plataformas como **Netlify** o **Vercel**. Contiene la configuración básica de Netlify en el archivo `netlify.toml`.
